## 1. Project Overview
This repository contains a comprehensive Explanatory Data Analysis (EDA) of the Olist E-commerce ecosystem in Brazil. Using a dataset of over 100,000 orders, this project identifies critical "leaks" in the customer journey, specifically focusing on how geography and shipping costs impact brand loyalty and review scores.

## 2. Business Context (Section 1)
Olist operates as a crucial intermediary for Brazilian SMEs. However, the diverse geography of Brazil—ranging from dense urban centers in São Paulo to the isolated Amazon—creates a "Logistics Crisis." This project provides the Senior Leadership Team (SLT) with data-driven strategies to transition Olist from a "Middle-Man" to a "Logistics Powerhouse."

## 3. Data Audit & Preprocessing (Sections 2 & 3)
* **Data Integrity:** A quantitative audit was performed to identify missing values and "Phantom" delivery dates.
* **Cleaning:** Non-informative columns were removed, and feature engineering was used to create new metrics like `delivery_days` and `ship_percent` (Freight Ratio).

## 4. Key Strategic Insights (Section 4)
The analysis answers 10 critical business questions, including:
* **The November Anomaly:** Managing the 300% Black Friday volume spike.
* **The Trust Threshold:** Identifying the 15-day limit before customer satisfaction collapses.
* **Geographic Gaps:** Highlighting the 3x delivery delay in the North (RR) vs. the South (SP).
* **Toxic Sellers:** Using a "Strike System" to identify high-volume/low-rating vendors.
